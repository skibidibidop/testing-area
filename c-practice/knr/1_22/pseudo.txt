Fold long input lines into 2 or more shorter lines
Lines will fold before a set column

Counter++ for every character
Make \n every 20 characters

If 20th character is blank
	print \n
	Reset counter
	Go to start of next word
	Count and print at start of next word


If 20th character is not blank
	Go back to start of word
	Print \n
	Reset counter
	Count and print at start of word
